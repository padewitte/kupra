Mongo Rest Camel

A simple REST interface for MongoDB build with Apache Camel. All the power of MongoDB is accessible with only a JVM on your server.

REST services are expose on 8667 port

Docs and in a near future console is accesible on port 9667

List all items of a collection

MRC allow you to list all item of a collection. URI is build with

http://127.0.0.1:8666/mrc/{collection name}

http://camel.apache.org/mongodb.html#MongoDB-findAll

Enregistrer un item

MRC permet de sauvegarder un item dans une collection.

Exemple

http://127.0.0.1:8666/mrc/save?collection=test&body={name%20:%20'hello'}
